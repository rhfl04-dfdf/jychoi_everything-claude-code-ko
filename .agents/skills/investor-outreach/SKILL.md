# 투자자 아웃리치

짧고, 개인화되고, 행동하기 쉬운 투자자 커뮤니케이션을 작성합니다.

## 활성화 시점

- 투자자에게 콜드 이메일 작성
- 소개 요청 블럽 초안 작성
- 미팅 후 또는 응답 없을 때 후속 연락
- 펀드레이징 진행 중 투자자 업데이트 이메일 작성
- 펀드 테제 또는 파트너 적합성에 따른 아웃리치 맞춤화

## 핵심 규칙

1. 모든 아웃바운드 메시지를 개인화하세요.
2. 요청의 허들을 낮게 유지하세요.
3. 형용사가 아닌 증거를 사용하세요.
4. 간결하게 유지하세요.
5. 어떤 투자자에게나 보낼 수 있는 일반적인 문구는 절대 보내지 마세요.

## 콜드 이메일 구조

1. 제목: 짧고 구체적으로
2. 오프너: 이 투자자를 선택한 이유
3. 피치: 회사가 무엇을 하는지, 왜 지금인지, 어떤 증거가 중요한지
4. 요청: 하나의 구체적인 다음 단계
5. 마무리: 이름, 역할, 필요 시 신뢰도를 높이는 한 가지 근거

## 개인화 소스

다음 중 하나 이상을 참조:
- 관련 포트폴리오 회사
- 공개된 투자 테제, 강연, 포스트, 아티클
- 공통 지인
- 투자자 집중 분야와 명확한 시장 또는 제품 적합성

해당 컨텍스트가 없는 경우 요청하거나, 초안이 개인화가 필요한 템플릿임을 명시하세요.

## 후속 연락 주기

기본 설정:
- 0일차: 최초 아웃바운드
- 4-5일차: 새로운 데이터 포인트 하나와 함께 짧은 후속 연락
- 10-12일차: 깔끔하게 마무리하는 마지막 후속 연락

사용자가 더 긴 시퀀스를 원하지 않는 한 이후에는 계속 연락하지 마세요.

## 소개 요청

소개해주는 사람이 편하게 할 수 있도록:
- 소개가 적합한 이유 설명
- 전달 가능한 블럽 포함
- 전달용 블럽은 100단어 이내로 유지

## 미팅 후 업데이트

포함 사항:
- 논의된 구체적인 내용
- 약속한 답변 또는 업데이트
- 가능하면 새로운 증거 포인트 하나
- 다음 단계

## 품질 기준

납품 전:
- 메시지가 개인화됨
- 요청이 명확함
- 군더더기나 애원하는 표현 없음
- 증거 포인트가 구체적
- 단어 수를 간결하게 유지
