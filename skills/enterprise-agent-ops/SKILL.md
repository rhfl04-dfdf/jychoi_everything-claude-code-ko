# 엔터프라이즈 에이전트 운영

단일 CLI 세션을 넘어 운영 제어가 필요한 클라우드 호스팅 또는 지속 실행 에이전트 시스템에 이 스킬을 사용하세요.

## 운영 도메인

1. 런타임 수명 주기 (시작, 일시정지, 중지, 재시작)
2. 관찰 가능성 (로그, 메트릭, 트레이스)
3. 안전 제어 (스코프, 권한, 킬 스위치)
4. 변경 관리 (롤아웃, 롤백, 감사)

## 기본 제어 항목

- 불변 배포 아티팩트
- 최소 권한 자격증명
- 환경 수준 시크릿 주입
- 하드 타임아웃 및 재시도 예산
- 고위험 작업에 대한 감사 로그

## 추적할 메트릭

- 성공률
- 작업당 평균 재시도 횟수
- 복구 시간
- 성공한 작업당 비용
- 실패 유형 분포

## 인시던트 패턴

실패가 급증할 때:
1. 새 롤아웃 동결
2. 대표적인 트레이스 수집
3. 실패 경로 격리
4. 최소한의 안전한 변경으로 패치
5. 회귀 + 보안 검사 실행
6. 점진적으로 재개

## 배포 통합

이 스킬과 함께 사용:
- PM2 워크플로우
- systemd 서비스
- 컨테이너 오케스트레이터
- CI/CD 게이트
