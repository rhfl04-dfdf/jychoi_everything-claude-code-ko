## 동영상 검색 및 하이라이트

### 하이라이트 릴 생성
긴 동영상 (컨퍼런스 강연, 강의, 회의 녹화)을 업로드하고, 주제별로 주요 순간을 검색하여 ("제품 발표", "Q&A 세션", "데모") 매칭 구간을 자동으로 공유 가능한 하이라이트 릴로 컴파일합니다.

### 검색 가능한 동영상 라이브러리 구축
동영상을 컬렉션에 일괄 업로드하고, 음성 단어 검색을 위해 인덱싱한 다음, 전체 라이브러리에서 쿼리합니다. 수백 시간의 콘텐츠에서 특정 주제를 즉시 찾습니다.

### 특정 클립 추출
쿼리에 매칭되는 순간을 검색하고 ("예산 논의", "액션 아이템") 각 매칭 구간을 자체 스트림 URL이 있는 개별 클립으로 추출합니다.

## 동영상 향상

### 전문적인 완성도 추가
원본 영상을 다음으로 향상시킵니다:
- 음성에서 자동 생성된 자막
- 특정 타임스탬프의 커스텀 썸네일
- 배경 음악 오버레이
- 생성된 이미지를 활용한 인트로/아웃트로 시퀀스

### AI 강화 콘텐츠
기존 동영상과 생성형 AI를 결합합니다:
- 전사에서 텍스트 요약 생성
- 동영상 길이에 맞는 배경 음악 생성
- 제목 카드 및 오버레이 이미지 생성
- 모든 요소를 세련된 최종 출력으로 믹싱

## 실시간 캡처 (데스크탑/회의)

### AI가 포함된 화면 + 오디오 녹화
화면, 마이크, 시스템 오디오를 동시에 캡처합니다. 실시간으로 얻을 수 있는 것:
- **실시간 전사** - 발생하는 음성을 텍스트로 변환
- **오디오 요약** - 주기적인 AI 생성 토론 요약
- **시각 인덱싱** - 화면 활동의 AI 설명

### 요약이 포함된 회의 캡처
모든 참가자의 실시간 전사로 회의를 녹화합니다. 실시간으로 전달되는 주요 토론 포인트, 결정 사항, 액션 아이템이 포함된 주기적 요약을 받습니다.

### 화면 활동 추적
AI 생성 설명으로 화면에서 일어나는 일을 추적합니다:
- "사용자가 Google Sheets에서 스프레드시트를 탐색 중"
- "사용자가 Python 파일이 있는 코드 에디터로 전환"
- "화면 공유가 활성화된 화상 통화"

### 세션 후 처리
캡처가 종료된 후 녹화는 영구 동영상으로 내보내집니다. 그 다음:
- 검색 가능한 전사 생성
- 녹화 내에서 특정 주제 검색
- 중요한 순간의 클립 추출
- 스트림 URL 또는 플레이어 링크로 공유

## 라이브 스트림 인텔리전스 (RTSP/RTMP)

### 외부 스트림 연결
RTSP/RTMP 소스 (보안 카메라, 인코더, 방송)의 라이브 동영상을 수집합니다. 실시간으로 콘텐츠를 처리하고 인덱싱합니다.

### 실시간 이벤트 감지
라이브 스트림에서 감지할 이벤트를 정의합니다:
- "제한 구역에 사람 진입"
- "교차로에서 교통 위반"
- "선반에 보이는 제품"

이벤트 발생 시 WebSocket 또는 웹훅을 통해 알림을 받습니다.

### 라이브 스트림 검색
녹화된 라이브 스트림 콘텐츠에서 검색합니다. 수 시간의 연속 영상에서 특정 순간을 찾고 클립을 생성합니다.

---

## 콘텐츠 모더레이션 및 안전

### 자동화된 콘텐츠 검토
AI로 동영상 장면을 인덱싱하고 문제 있는 콘텐츠를 검색합니다. 폭력, 부적절한 콘텐츠, 정책 위반이 포함된 동영상에 플래그를 지정합니다.

### 비속어 감지
오디오에서 비속어를 감지하고 위치를 파악합니다. 선택적으로 감지된 타임스탬프에 삐 소리 오버레이를 추가합니다.

---

## 플랫폼 통합

### 소셜 미디어 형식 지정
다양한 플랫폼에 맞게 동영상을 리프레임합니다:
- TikTok, Reels, Shorts용 세로 (9:16)
- Instagram 피드용 정사각형 (1:1)
- YouTube용 가로 (16:9)

### 전달을 위한 트랜스코딩
다양한 전달 목표에 맞게 해상도, 비트레이트, 품질을 변경합니다. 웹, 모바일, 방송에 최적화된 스트림을 출력합니다.

### 공유 가능한 링크 생성
모든 작업은 재생 가능한 스트림 URL을 생성합니다. 웹 플레이어에 임베드하거나, 직접 공유하거나, 기존 플랫폼과 통합합니다.

## 워크플로 요약

| 목표 | VideoDB 접근 방식 |
|------|------------------|
| 동영상에서 순간 찾기 | 음성 단어/장면 인덱싱 → 검색 → 클립 컴파일 |
| 하이라이트 생성 | 여러 주제 검색 → 타임라인 구성 → 스트림 생성 |
| 자막 추가 | 음성 단어 인덱싱 → 자막 오버레이 추가 |
| 화면 + AI 녹화 | 캡처 시작 → AI 파이프라인 실행 → 동영상 내보내기 |
| 라이브 스트림 모니터링 | RTSP 연결 → 장면 인덱싱 → 알림 생성 |
| 소셜 미디어용 형식 변환 | 목표 종횡비로 리프레임 |
| 클립 결합 | 여러 에셋으로 타임라인 구성 → 스트림 생성 |
