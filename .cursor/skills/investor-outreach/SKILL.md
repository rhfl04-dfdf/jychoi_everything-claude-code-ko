# 투자자 아웃리치

짧고, 개인화되며, 행동하기 쉬운 투자자 커뮤니케이션을 작성합니다.

## 활성화 시점

- 투자자에게 콜드 이메일 작성
- 따뜻한 소개 요청 초안 작성
- 미팅 후 또는 응답 없을 때 팔로업 발송
- 과정 중 투자자 업데이트 이메일 작성
- 펀드 투자 방향 또는 파트너 적합성 기반 아웃리치 맞춤화

## 핵심 규칙

1. 모든 아웃바운드 메시지를 개인화하세요.
2. 요청을 마찰 없게 유지하세요.
3. 형용사가 아닌 증거를 사용하세요.
4. 간결하게 유지하세요.
5. 어느 투자자에게나 보낼 수 있는 일반적인 카피는 절대 보내지 마세요.

## 콜드 이메일 구조

1. 제목 줄: 짧고 구체적으로
2. 도입: 이 투자자를 선택한 이유
3. 피치: 회사가 하는 일, 지금인 이유, 중요한 증거
4. 요청: 하나의 구체적인 다음 단계
5. 서명: 이름, 역할, 필요한 경우 신뢰도 근거 하나

## 개인화 소스

다음 중 하나 이상을 참조하세요:
- 관련 포트폴리오 회사
- 공개 투자 방향, 발표, 게시물, 또는 기사
- 공동 연결
- 투자자의 초점 분야와 명확한 시장 또는 제품 적합성

해당 맥락이 없으면, 요청하거나 초안이 개인화를 기다리는 템플릿임을 명시하세요.

## 팔로업 주기

기본값:
- 0일: 초기 아웃바운드
- 4-5일: 새로운 데이터 포인트 하나를 포함한 짧은 팔로업
- 10-12일: 깔끔한 마무리로 최종 팔로업

사용자가 더 긴 시퀀스를 원하지 않는 한 그 이후에는 계속 재촉하지 마세요.

## 따뜻한 소개 요청

연결해주는 사람의 일을 쉽게 만들어주세요:
- 소개가 왜 적합한지 설명하세요
- 전달 가능한 블럽을 포함하세요
- 전달 가능한 블럽은 100단어 이하로 유지하세요

## 미팅 후 업데이트

포함 내용:
- 논의된 구체적인 내용
- 약속한 답변 또는 업데이트
- 가능하면 새로운 증거 포인트 하나
- 다음 단계

## 품질 기준

전달 전:
- 메시지가 개인화되어 있음
- 요청이 명시적임
- 군더더기나 구걸하는 언어 없음
- 증거 포인트가 구체적임
- 단어 수가 타이트하게 유지됨
