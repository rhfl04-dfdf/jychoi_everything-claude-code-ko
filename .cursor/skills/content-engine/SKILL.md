# 콘텐츠 엔진

하나의 아이디어를 어디서나 같은 내용을 올리는 대신, 각 플랫폼에 최적화된 강력한 콘텐츠로 전환합니다.

## 활성화 시점

- X 게시물 또는 스레드 작성
- LinkedIn 게시물 또는 런치 업데이트 초안 작성
- 숏폼 영상 또는 YouTube 해설 스크립트 작성
- 기사, 팟캐스트, 데모, 문서를 소셜 콘텐츠로 재활용
- 런치, 마일스톤, 특정 주제를 중심으로 가벼운 콘텐츠 계획 수립

## 첫 번째 질문

다음을 명확히 하세요:
- 소스 자료: 무엇을 기반으로 변환할 것인가
- 대상 독자: 빌더, 투자자, 고객, 운영자, 또는 일반 대중
- 플랫폼: X, LinkedIn, TikTok, YouTube, 뉴스레터, 또는 멀티플랫폼
- 목표: 인지도, 전환, 채용, 권위 확립, 런치 지원, 또는 참여 유도

## 핵심 규칙

1. 플랫폼에 맞게 조정하세요. 같은 카피를 여러 곳에 동시 게시하지 마세요.
2. 요약보다 훅이 더 중요합니다.
3. 모든 게시물은 하나의 명확한 아이디어를 담아야 합니다.
4. 슬로건보다 구체적인 내용을 사용하세요.
5. 요청은 작고 명확하게 유지하세요.

## 플랫폼별 가이드

### X
- 빠르게 시작하세요
- 게시물 또는 스레드 내 각 트윗마다 하나의 아이디어
- 필요하지 않은 경우 본문에 링크를 넣지 마세요
- 해시태그 남용을 피하세요

### LinkedIn
- 강력한 첫 번째 줄
- 짧은 단락
- 교훈, 결과, 핵심 내용에 대한 더 명시적인 프레임

### TikTok / 숏폼 영상
- 처음 3초가 시선을 사로잡아야 합니다
- 내레이션만이 아닌 비주얼 중심으로 스크립트 작성
- 데모 하나, 주장 하나, CTA 하나

### YouTube
- 결과를 초반에 보여주세요
- 챕터별로 구성하세요
- 20-30초마다 시각적 변화를 주세요

### 뉴스레터
- 관련 없는 항목들의 묶음이 아닌 하나의 명확한 관점을 전달하세요
- 섹션 제목을 훑어보기 쉽게 만드세요
- 도입 단락이 실질적인 역할을 하도록 유지하세요

## 재활용 흐름

기본 캐스케이드:
1. 앵커 자료: 기사, 영상, 데모, 메모, 또는 런치 문서
2. 3-7개의 원자적 아이디어 추출
3. 플랫폼별 변형 작성
4. 결과물 간 중복 제거
5. 플랫폼 목적에 맞게 CTA 정렬

## 결과물

캠페인 요청 시 반환:
- 핵심 각도
- 플랫폼별 초안
- 선택적 게시 순서
- 선택적 CTA 변형
- 게시 전 필요한 누락 입력값

## 품질 기준

전달 전:
- 각 초안이 해당 플랫폼에 맞게 자연스럽게 읽히는지 확인
- 훅이 강력하고 구체적인지 확인
- 일반적인 과장 언어 없음
- 요청하지 않은 경우 플랫폼 간 중복 카피 없음
- CTA가 콘텐츠와 대상 독자에 맞는지 확인
