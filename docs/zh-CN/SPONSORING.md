# ECC 후원

ECC는 Claude Code, Cursor, OpenCode 및 Codex 애플리케이션/CLI에서 유지되는 오픈소스 agent 성능 테스트 시스템입니다.

## 후원 이유

후원은 다음에 직접 자금을 지원합니다:

* 더 빠른 버그 수정 및 릴리스 주기
* 크로스 테스트 플랫폼의 플랫폼 일관성 작업
* 커뮤니티에 무료로 제공되는 공개 문서, skill 및 안정성 도구

## 후원 등급

이것은 실용적인 출발점이며, 협력 범위에 따라 조정할 수 있습니다.

| 등급 | 가격 | 최적 대상 | 포함 내용 |
|------|-------|----------|----------|
| 파일럿 파트너 | $200/월 | 첫 후원 협력 | 월간 메트릭 업데이트, 로드맵 미리보기, 우선 유지관리자 피드백 |
| 성장 파트너 | $500/월 | ECC를 적극 채택하는 팀 | 파일럿 혜택 + 월간 오피스 아워 동기화 + 워크플로우 통합 가이드 |
| 전략 파트너 | $1,000+/월 | 플랫폼/생태계 파트너 | 성장 혜택 + 릴리스 조율 지원 + 더 깊은 유지관리자 협업 |

## 후원 보고서

매월 공유되는 메트릭은 다음을 포함할 수 있습니다:

* npm 다운로드 수 (`ecc-universal`, `ecc-agentshield`)
* 저장소 채택 현황 (star, fork, 기여자)
* GitHub 앱 설치 추세
* 릴리스 주기 및 안정성 마일스톤

정확한 명령어 스니펫 및 재현 가능한 풀 프로세스는 [`docs/business/metrics-and-sponsorship.md`](../business/metrics-and-sponsorship.md)를 참조하세요.

## 기대치 및 범위

* 후원은 유지보수를 지원하고 가속화합니다; 프로젝트 소유권을 이전하지 않습니다.
* 기능 요청은 후원 등급, 생태계 영향 및 유지보수 리스크에 따라 우선순위가 매겨집니다.
* 보안 및 안정성 수정이 완전히 새로운 기능보다 우선합니다.
