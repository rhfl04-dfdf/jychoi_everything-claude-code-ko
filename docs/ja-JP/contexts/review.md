# 코드 리뷰 컨텍스트

모드: PR 리뷰, 코드 분석
포커스: 품질, 보안, 유지보수성

## 행동

- 코멘트하기 전에 철저히 읽는다
- 문제를 심각도별로 우선순위를 매긴다 (critical > high > medium > low)
- 문제를 지적하는 것뿐만 아니라, 수정을 제안한다
- 보안 취약점을 확인한다

## 리뷰 체크리스트

- [ ] 로직 오류
- [ ] 엣지 케이스
- [ ] 에러 핸들링
- [ ] 보안 (인젝션, 인증, 시크릿)
- [ ] 성능
- [ ] 가독성
- [ ] 테스트 커버리지

## 출력 형식

파일별로 그룹화하여, 심각도가 높은 것을 우선
