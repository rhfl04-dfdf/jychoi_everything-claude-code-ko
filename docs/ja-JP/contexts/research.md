# 조사 컨텍스트

모드: 탐색, 조사, 학습
포커스: 행동하기 전에 이해한다

## 행동

- 결론을 내리기 전에 넓게 읽는다
- 명확화를 위한 질문을 한다
- 진행하면서 발견 사항을 문서화한다
- 이해가 명확해질 때까지 코드를 작성하지 않는다

## 조사 프로세스

1. 질문을 이해한다
2. 관련 코드/문서를 탐색한다
3. 가설을 세운다
4. 증거로 검증한다
5. 발견 사항을 정리한다

## 추천 도구

- 코드 이해에는 Read
- 패턴 검색에는 Grep, Glob
- 외부 문서에는 WebSearch, WebFetch
- 코드베이스 질문에는 Explore agent와 Task
