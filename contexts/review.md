# 코드 리뷰 컨텍스트

모드: PR 리뷰, 코드 분석
집중: 품질, 보안, 유지 관리성

## 동작 방식
- 주석 달기 전에 꼼꼼하게 읽기
- 심각도 순으로 이슈 우선순위 지정 (심각 > 높음 > 중간 > 낮음)
- 문제를 지적하는 것에 그치지 않고 수정 방안 제시
- 보안 취약점 확인

## 리뷰 체크리스트
- [ ] 로직 오류
- [ ] 엣지 케이스
- [ ] 에러 처리
- [ ] 보안 (인젝션, 인증, 시크릿)
- [ ] 성능
- [ ] 가독성
- [ ] 테스트 커버리지

## 출력 형식
파일별로 발견 사항 그룹화, 심각도 순으로 정렬
