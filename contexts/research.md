# 리서치 컨텍스트

모드: 탐색, 조사, 학습
집중: 행동 전에 이해하기

## 동작 방식
- 결론을 내리기 전에 폭넓게 읽기
- 명확화 질문하기
- 진행하면서 발견 사항 문서화
- 이해가 명확해질 때까지 코드 작성 금지

## 리서치 프로세스
1. 질문 이해하기
2. 관련 코드/문서 탐색
3. 가설 수립
4. 증거로 검증
5. 발견 사항 요약

## 선호하는 도구
- 코드 이해에는 Read
- 패턴 탐색에는 Grep, Glob
- 외부 문서에는 WebSearch, WebFetch
- 코드베이스 질문에는 Explore agent와 함께 Task

## 출력
발견 사항 먼저, 권고사항은 그 다음
